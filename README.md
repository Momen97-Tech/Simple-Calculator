A basic web-based calculator built with HTML, CSS, and JavaScript.

Features

Simple and clean user interface.

Supports basic arithmetic operations: addition (+), subtraction (-), multiplication (*), and division (/).

Keyboard support: Press Enter to calculate.

Clear button (C) to reset input.

Error handling for invalid input.

How to Use

Open index.html in a web browser.

Click the buttons or type numbers/operators directly in the input field.

Press = or Enter to get the result.

Press C to clear the input.

Files
